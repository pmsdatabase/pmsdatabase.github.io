2023/07 発狂PMS難易度表更新再開に伴う当サイトの変更点・ガイドラインまとめ

このたび、発狂PMS難易度表の更新が再開される ことになりました。
再開にあたって管理者の方から PMS Database にデータ提携の打診があり、元々管理されていたものを引き継いでいる立場ですので これをお受けしました。

リニューアルにあたっては PMS Database の一部ページやルールの改訂 (擦り合わせ) を行い、2023/07/22 にサイト反映が完了しています。
このページでは、改訂の具体的な内容と新しいルールについて、ガイドライン形式で解説します。


発狂PMS難易度表 / PMS Database のサイト統合

■ index.html (現在の Home) は、PMS Database / 発狂PMS難易度表 に移動するためのポータルページに変更となりました。
　　・PMS Database としてのTOPページは　https://pmsdifficulty.xxxxxxxx.jp/home.html　に変更となりますので、必要に応じてブックマークの更新をお願いします。
　　・難易度表を含むその他のURLは変更していません。

Normal Records / Insane Records の仕様変更

■ 一部の難易度フォルダを改訂
　　・Insane Records に「P●0」フォルダを追加します。
　　　　・P●0 は発狂PMS難易度表の●0に対応していて、選抜された 44～45+ (と ●1降格) が含まれます。
　　　　・表外の譜面に関してはこれまでと同じく非発狂側 44, 45, 45+ フォルダに格納されます。
　　・Excluded Records にいくつかのフォルダを新設し、非常にセンシティブな表現が問題になった作品を再仕分けします。


■ 難易度査定について、LR2 / beatoraja 優位性の見直し
　　・新規追加譜面は、beatoraja からの査定意見を優先します。　*他本体からの意見も参考に取り入れます。アケコン / KB に近い関係と解釈してください。
　　・難易度移動提案は、beatoraja からの提案をお願いします。当面の間は Level Review ではなく 難易度変更提案所 を利用して提案してください。
　　　　・発狂PMS難易度表の登録譜面は、PMS Database 側からの難易度移動提案を不可とします。(更新内容を同期するため）
　　　　・発狂PMS難易度表で取り下げ/削除となった譜面は、難易度を維持したまま PMS Database に移動します。数字変更を希望される場合は別途提案してください。

■ PDB本表 "memo" (登録先難易度表種別) カラムの表示内容と色分けの変更
　　・「発狂PMS」 は発狂PMS難易度表の更新内容と同期します。
　　・「旧案内所」「見てない難易度」 は統合し、表示削除・色分け無し に変更となります。
　　・2023/07 以降の追加譜面には、新たに 「統合以降」 という表示を追加します。

■ PDB本表 "コメント" の内容修正
　　・管理者さんからの要望により、発狂PMS、旧案内所登録譜面に掲載されている所感コメントを削除します。
　　・Normal Record を中心に、高TOTAL譜面などの形骸化している一部表示を省略、掲載する情報の整理を行います。
　　　　・ただし、これらの内容はログ(後述) からは引き続き参照できるようにしておきます。アーカイブとして利用してください。

PMS Database のコンテンツ整理

■ PMS Database で取り扱っている一部難易度表について、2023/07 上半期更新後 時点のログを公開　Others > サブ難易度表
　　・取得する難易度表
　　　　・Normal Records / Insane Records
　　　　・発狂PMS難易度表 / 準発狂PMS難易度表 / 表外高難易度譜面案内所(旧案内所) / たんぱつ難易度表
　　　　　　・(2023/07/22) 「旧案内所」の内容やコンセプトは PMS Database が踏襲しているため、バックアップの作成は見送りました。

　　・難易度、曲コメントなどは全て統合前 (= LR2査定) の状態を維持しますが、今後の更新には対応しません。
　　・BOF差分企画の年度別難易度表など、既に更新停止となっている部分は現状維持とし、ログは取得しません。

■ 発狂PMS関連避難所(仮) を削除
　　・旧発狂PMS難易度表サイト(シュレーディンガーの猫) が復旧したため、紛らわしさ回避のため非公開に変更します。

■ PMS Database で使用中のスプレッドシート「案内所 (避難所)」は、名称を「PMS Database 関連作業所」と改めて利用継続
　　・PMS Database 関連作業所
　　・原則として、旧発狂PMS難易度表サイト(シュレーディンガーの猫) から誘導されるスプレッドシートは今後も使用しません。

■ PMS Database Grade (PDBG) コースの改訂・討論会について
　　・2025年度を目途にコースの改訂を予定していましたが、発狂PMS難易度表側で段位認定を制作される可能性が出てきたため、コースの扱い全般を見直します。
　　・方針が固まり次第、Discord Server で連絡します。

発狂PMS難易度表 / PMS Database の登録基準(レギュレーション) について

今後とも、発狂PMS難易度表 と PMS Database は 独立したレギュレーションで運営していきます。
諸般の理由から発狂PMS難易度表で取り下げられた譜面についても、PMS Database の場合は引き続き取り扱う、といった具合です。(細かい部分の解釈は各運営の一存とします。)

もし PMS Database からも取り下げをしなければならない場合は、二度手間で恐縮ですが、本人確認情報を添えて PMS Database のお問い合わせフォーム よりご連絡ください。
発狂PMS難易度表に関する質問・依頼等は、発狂PMS難易度表のお問い合わせフォーム を利用してください。
