# BOF PMS Package Releases

BOF関連のPMS差分パッケージについて、GitHub Releases上の固定タグとRelease assetsをまとめた一覧です。

## 配布方針

- **PMS差分パッケージの本公開所**：GitHub Releases
- **PMS差分パッケージの予備ミラー**：[Google Drive（BOF PMS Appends 2013-）](https://drive.google.com/drive/folders/1FVIT3iMx0TKklSZ3LNWh_MDDyX-zoDf_)
- **本体BMSパッケージ**：[Google Drive（BOF Packages 2004-）](https://drive.google.com/drive/folders/1R8T6s8es02eJczB9ykg0Zrr5DYt6oCKl)
- 本体BMSパッケージおよび本体BMSパッケージ用の修正差分は、GitHub Releasesには収録していません。
- 通常は各年度の`FullPackage`を使用してください。分割版、旧版利用者向け修正差分、関連ファイルは各Releaseページから取得できます。
- Gitのclone／forkだけではRelease assetsは複製されません。別環境へ再展開する場合は、下記の固定URLからassetsを別途取得してください。
