コースの概要

■ コースの全容をまとめたスプレッドシート (PMS Database 関連アンケート場)

■ 従来の ○段～ 形式ではなく、各難易度ごとのコース構成になっています。
■ 上下コース間のバランスを最優先に考えて、余力がある場合は 第五期発狂PMS段位認定 と相互練習になるようにデザインしました。
■ 各コースはレベル内で上位の譜面を中心に総合的に選曲しており、「段位ゲージでクリア出来れば、そのレベルの譜面をある程度掌握できる程度」の地力指標を目指しています。

コースの詳細な仕様について（クリックで開閉）

Normal Records

■ 複数のレベルを統合したコースがあり、レベル45に近づくにつれて刻みが細かくなります。
■ ○6 (45+) コースの難易度は ●01 コースとやや重複します。


　

Insane Records（～P●15）

■ 1レベルごとに1コースで構成されています。
■ 他の領域に比べて発狂段位が充実しているため、相互練習向けのコンセプトを重視しています。


　

Insane Records（P●16～）

■ 発狂PMS皆伝、Overjoy、Undefined へのステップアップ・相互練習に向くように細分化しています。
■ 細分化は ver.2022.08 公開時点の構成です。

　

LR2 / beatoraja の独自機能について

■ GLAssist ver.232 の段位タブ表示 (LR2)

■ Mocha-Repository の Class 表示 (beatoraja)
第五期 発狂PMS段位認定 と同様に 取得済段位を Class 表示できます。Mocha-Repository に送信の上、プロフィール画面から設定してください。シンボルは P○ / P● です。

■ EXTRA OPTIONS コース (beatoraja)
このページを難易度表として読み込んだ場合に限り、特殊オプションの追加された高難易度コースが選択可能です。
クリアデータ・IRランキングはそれぞれ独自に反映されます。スキモノな方々に向けた、限界へのチャレンジを。

オプションは現在のところ3種類:　NO GOOD (GOODがBADに!) / NO GREAT (COOL or BAD!) / NO SPEED (HS-FIXを無視し NORMAL HS1.0倍に固定)


レギュレーション等

■ PMS Database 本表で扱っている全てのPMS差分(BMSONは不可) が選曲対象です。ただし R18(-G) や再生不良等で個別に考慮する場合があります。
■ 本表と同じくAC準拠のコントローラーでのプレイを想定していますが、KBでのプレイで難易度差が極端になるものは考慮します。(特に低難易度)
■ BPM変化、LN、地雷ノート等の個人差要素について制限は設けませんが、LR2 の MAX-FIX , beatoraja の MAIN-BPM FIX機能で表示が大きく崩れるギミックは不採用とします。
■ beatoraja の段位認定用カスタマイズ項目は「GRADE_MIRROR」「GAUGE_LR2」「LN」を適用します。トロフィー機能には対応していません。

■ クリア難易度と段位ゲージの難易度は必ずしも一致しないため、同一レベルフォルダ内での選曲制限は設けません。(上位/下位レベルから選曲する場合があります)
■ 発狂PMS難易度表の登録差分や、同一の差分作者さんで構成が偏りすぎないように注意します。過去の段位認定で使用された差分もある程度考慮します。

注意事項

■ 使用させて頂いているPMSは段位認定としての使用許諾を得たものではありませんので、コースセット向けのパッケージ配布等には対応できません。
■ 予告なく公開を取り下げる場合があります。予めご了承ください。
