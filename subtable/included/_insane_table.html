同梱PMS譜面一覧表 (発狂)

この難易度表について

この表は、BMSイベントやパッケージなどに同梱されたPMS譜面を絞り込むための難易度表です。導入にかかる工数が比較的少なく済むので、収集の足掛かりに使えるかもしれません。
PMS限定イベント・パッケージについてはパッケージでの収集率が非常に高く、探す余地もなくフォルダから直接選曲できるため、暫定で掲載対象外としています。
PMS Database メイン表の定例更新時にこの表も更新されます。

掲載対象外としたもの：
　- Toy Musical / Toy Musical 2
　- Colorful Canvas / Colorful Channel
　- SPRING 9KEYS ROAD 2011 / SUMMER 9KEYS FESTIVAL 2012 / AUTUMN 9KEYS STORY 2013 / WINTER 9KEYS SPARKLE 2015
　- SPRING 9KEYS BLOSSOM 2020

PMSパッケージの側面があるものの、掲載対象としているもの：
　- キューキービーツ　(非常に古いイベントで安定したパッケージ入手が難しく、入手先情報が散逸しているため)
　- PrimiTiv など全曲に同梱PMSが含まれるBMSパッケージ　(PMSが主体ではなく、多彩なプレイスタイルを備えたBMSパッケージ、というのが本旨のため)
　- BOFPMS差分企画 に登録されている同梱PMS譜面
