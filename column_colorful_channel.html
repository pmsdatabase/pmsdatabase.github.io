コラム - PMSパッケージ "Colorful Channel" 導入ガイド &コンテンツ紹介

Colorful Channel official website ++ download　 第3回放送 フルセット (CC + CC2 一括)

Colorful Canvas (CC) / Colorful Channel (CC2) は、～2013年にかけて更新されていた PMSパッケージ企画です。
(2020/01/01) Colorful Channel 企画終了が正式に発表されました が、パッケージは現在も公式サイトからダウンロードすることができます。

しかし、一部のコンテンツは専用のアプリケーションを使った解禁が必要となっており、これから追いかけるのはやや大変かもしれません。
そこでこのコラムでは、解禁方法のアナウンス、ならびに周辺コンテンツについてご紹介させて頂きます。
情報掲載にあたってはパッケージ管理者の方から許諾を頂いていますが、ネタバレ要素を含みますので、注意して参照してください。

パッケージの概要

■ Bms > 「Colorful Canvas」「Colorful Channel」内の各フォルダ
本体PMSが格納されています。LR2 / beatoraja で再生する場合はこの中身を登録してください。
ただし！ 解禁した隠し曲はここに追加されていくので、フォルダ整理する場合は要注意です。 DL直後にあらかじめ解禁作業を行ってしまうのも良いでしょう。

■ sch.exe (シ～クレットチャンネル)
「リクエスト」でパスワードを入力し、「解禁状況」に出現したバナーをクリックすると Bms フォルダ内に楽曲が増えます。

■ User > Char > 「_pmcc」「_pmcc2」内の各フォルダ
本家さんのように動くマスコットキャラクター「ぽみゅキャラ」が多数格納されています。beatoraja の一部スキンが再生対応しています。
　- PeacefulPlay - 画像表示・ぽみゅキャラ表示　(2026/06 現在、beatoraja 9KEYS でぽみゅキャラに対応した代表的なプレイスキン)
　- Pomyu Chara Archive / ぽみゅキャラアーカイブ　(with English Guide)
　- beatorajaでのぽみゅキャラの使い方　(汎用的で詳細な導入解説記事)

----------
■ Fps.exe / FpsStart.exe
同梱のPMSプレイヤー「ふぃーりんぐぽみゅせかんど」の本体です。使ってみたい場合は how to play を参照してください。
　- Fps.exe を普通に起動できた場合は、記事内の初手 .NET Framework の導入は不要です。
　- 初回起動で躓いた場合は、FpsStart.exe を開いてデフォルトスキンを選択 > [設定] > 裏みゅにCS の設定画面で [保存] > [終了]　次回以降は Fps.exe から起動できます。
　- 全画面表示がうまくいかない場合は、スケーリング設定を変更すると改善されることがあります。
　　　- 筆者は Windows 10 / 11 (64bit) 各環境下で、DisplayChange の設定 640*480 / 32BitColor / 60Hz で起動確認しました。
