この難易度表について

事情により本表に登録していないPMS差分をリストしています。
未登録の差分と区別するために使用しています。難易度表ツールに対応していますが、登録は自己責任でお願いします。


分類について

■要調査 … 公開・非公開がはっきりしていない差分です。下記「入手困難」と合わせて情報提供をお願いします。
■入手困難 … 情報をお持ちの方おられましたら、ぜひ PMS Database 関連作業所 に一報ください。
■キー音無し … (BMSとしての意図を持って公開された) キー音無しBMS作品とその差分です。
■18keys(DPPMS) … ポプコンが2台必要なPMS差分です。ナナシグルーヴ や ふぃーりんぐぽみゅせかんど が再生対応しています。
■大人の事情 … 色々あるものたちです。(一部コメントに追記あり)
■R18 / 下ネタ / グロゴア / ジャンプスケア / センシティブ / … 詳細やご意見は NSFW(再生注意) List & P† Discussion 　- PMS Database 関連アンケート場 に記入ください。
■取り下げ … 作者さんの要望により本表から撤回した差分です。特性上、URLは掲載していません。
■修正あり … ミス修正のほか、配置リニューアルがあったもの (いわゆる旧譜面) を含みます。公開停止となっている場合、URLは記載しません。
■マイナーチェンジ … 複数の入手経路があり、かつ難易度やBGA定義などの細かい修正により md5 が異なっている差分のうち、本表に掲載していないもののリストです。

■その他 … 上記以外で LR2IR にデータが存在する全ての差分のリストです。(不定期に走査しています)
