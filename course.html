コースデータ案内所 - PMS Database

UPDATE: 2026/06/20

主に LR2IR で配布されていたコースデータを広く収集し、平均レベル準拠の難易度表形式でリストしました。このページの URL は、難易度表ツールへの登録に対応しています。
色分け：黄色は PMS段位認定 2024 の課題コース、オレンジ色は PMS Database Grade Courses ver.2022.08 の課題コースです。


重要なお知らせ

LR2IR の閉鎖、ならびに beatoraja で作成したコースを Mocha-Repository から直接ダウンロードできるようになるアップデート があったことなどを総合的に踏まえ、
2026年6月をもちまして、コース追加のリクエストをはじめ、コースデータ案内所全体の更新を停止することにしました。
今後このページは「LR2IR に存在したコースデータを beatoraja 用に引き継いだアーカイブ集」として、必要に応じてご利用いただければと思います。

■ 関連作業用のスプレッドシート跡地（PMS Database 関連アンケート場）


更新履歴

2026/06/20　データの刷新を行いました。
　・ 各コースの LR2IR リンクを魚拓サイトの URL へ暫定的に変更しました。また、コースファイル単品のダウンロードリンクはリンク切れのため削除しました。
　・ PMS段位認定 2024 と PMS Database Grade Courses ver.2022.08 の課題コースを追加しました。
　・ 今回の更新をもって、旧 LR2IR 内に残っていたすべての 9KEYS コースデータの精査が完了しました。これらの更新分のみを一括導入する LR2 用コースファイルは こちら です。
　・ 各譜面の難易度とコースの平均難易度を最新の値に更新しました。ただし P●0 には複数の難易度が混在しているので、もともとの難易度表記（PLv）を優先しています。


利用方法（クリックで開閉）

LR2 と beatoraja で利用方法や利用できる機能が異なります。

LR2 を利用している場合

コースファイルをダウンロードし、LR2SETUP 画面へドラッグ＆ドロップして登録した後、LR2 の選曲画面で COURSE/NONSTOP/段位認定 フォルダから選曲します。
コースファイルは、下記のまとめファイルから一括登録できます。
登録済みのコースファイルと重複した場合は、選曲画面の TAG EDITOR > DELETE ボタンで削除できます（この操作は既存のクリアランプに影響しません）。

GLAssist（v232 / v135） でこのページの URL を読み込むと、各コースのクリア状況（v135）や最新の段位取得状況（v232）を確認できます。
仕様により、未プレイの段位は所持していても NO SONG と表示されます。また、カスタムフォルダ内からの選曲はできません。

beatoraja を利用している場合

このページの URL を難易度表リソースとして登録すると、カスタムフォルダ内にコースリストが出力されますので、そこから選曲してください。
レベル別の選曲フォルダは表示用のダミーで、選曲には使用できません。
